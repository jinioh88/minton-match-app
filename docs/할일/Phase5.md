### 진행 메모 (2026-04-28)

- 채팅/알림 REST 연동용 DTO·API 서비스·리포지토리 추가.
- 채팅 탭에 채팅방 목록 및 알림 배지 연결.
- 채팅 목록을 진행/지난 매칭으로 분리하고 매칭 상태 태그를 표시.
- 채팅방 화면에 상단 매칭 요약, 메시지 목록/입력, `afterId` 기반 5초 폴링 반영.
- 채팅방 과거 메시지 `cursor` 조회(이전 메시지 불러오기/상단 스크롤 자동 로딩) 반영.
- 채팅방 목록에 내 상태 태그(방장·참여확정·대기중·참석제안) 표시.
- 매칭 상세에서 채팅 진입 버튼 추가, `GET /api/matches/{matchId}/chat` 기반 진입 경로 연결.
- 알림 센터(`GET` 목록, 단건 읽음, 전체 읽음, 관련 매칭 이동) 추가.
- 알림 타입별 기본 카피/아이콘을 반영하고, 일부 타입(`PARTICIPATION_ACCEPTED`, 대기열 계열)은 알림에서 채팅방으로 바로 이동.

### 비고

- 채팅방별 읽지 않음 개수(`unread_count`)는 Sprint5 채팅방 목록 응답 스키마에 없어, 현재는 전체 미읽음 알림 배지만 노출.
